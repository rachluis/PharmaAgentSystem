# 根据 README 文件，三个核心 CSV 文件：

## General Payments (一般支付)

内容： 包含与研究无关的费用，例如咨询费、演讲费、餐饮费、差旅费等 。

价值 (高)： 这是你进行“医生商业价值画像”的核心数据。你可以通过这些数据分析出哪些医生是“KOL（意见领袖）”（拿了很多演讲费），哪些医生关系维护得好（餐饮和差旅多）。

## Research Payments (研究支付)

内容： 与研究协议或研究方案相关的付款 。

价值 (中)： 用于区分“学术型医生”和“商业型医生”。如果一个医生这类金额很高，说明他在学术界有影响力。

## Ownership/Investment (持股/投资)

内容： 医生在药企的持股或投资情况 。

价值 (低)： 对于本科毕设可能过于复杂，初期可以先忽略，专注于前两类。

训练可行性： 你可以提取字段（如：医生ID、总金额、支付次数、支付类别），输入 K-Means 模型，很容易将医生聚类为：“高价值KOL”、“学术专家”、“普通医生”等群体。
